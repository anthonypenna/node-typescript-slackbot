# Node Typescript Template

Starter template for developing Node.js applications with Typescript.

Includes `Jest`.

## Commands

**Run unit tests**

```sh
$ npm run test
```

**Run unit tests in watch mode**

```sh
$ npm run test:watch
```

**Run application in development mode**

```sh
$ npm run serve
```

**Run application in production mode**

```sh
$ npm run test:watch
```
